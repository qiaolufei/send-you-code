# 易代码

> 前端采用Vue框架，后端采用Spring Boot框架实现，一款用户可以自由发单、接单的程序员有偿互助平台

#### 目前具有的功能
- [x] 注册、登录、注销
- [x] 首页跳转第三方平台 
- [x] 发布需求
- [x] 按需浏览查找需求任务
- [x] 双方沟通交流
- [x] 编辑个人信息
- [x] 浏览历史发布需求
